# AI Customer Service

A learn-n-build project for mastering MERN stack, Natural Language Models (NLM), and Hugging Face integration. This project is designed for continuous learning and improvement.

## Features

- React frontend (Vite)
- Chat UI with emoji support
- Modular component structure
- Planned Backend integration (Mongodb, Nodejs, Expressjs) & NLM integration using Hugging Face

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```
